Attendance Automation Bot using Automation Anywhere Software

This bot automates the complete attendance processing pipeline using Automation Anywhere. It reads raw Excel sheets submitted by employees, formats the data, calculates absences and working hours, generates summaries, and emails reportsâ€”all without manual intervention.

Problem It Solves

Manual attendance tracking is time-consuming and error-prone. This bot eliminates the need for manual data handling, reducing errors and increasing HR productivity.

Features

  • Reads attendance Excel files from a predefined folder
  • Extracts and validates date-wise in/out entries for each employee
  • Calculates total working hours and flags absences/irregularities
  • Generates weekly summary Excel reports
  • Auto-sends reports via email to concerned managers

Technologies Used

Automation Anywhere, Excel Automation, Email Automation, Bots Insight Dashboard
